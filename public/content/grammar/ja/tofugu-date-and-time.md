# Date and Time

> **Source:** Content from [Tofugu](https://www.tofugu.com/japanese-grammar/date-and-time/). For personal study.

The date format in Japan is **Year → Month → Day → (Weekday)**, from most general to most specific. Time uses 〜時 (hour), 〜分 (minute), and 〜秒 (second).

## Date

### 〜年 (ねん) — Year
Used with numbers: 2024年 (にせんにじゅうよねん). Japan also uses the imperial calendar (e.g., 令和6年 = 2024).

### 〜月 (がつ) — Month

| Japanese | Reading | Month |
|---|---|---|
| 一月 | いちがつ | January |
| 二月 | にがつ | February |
| 三月 | さんがつ | March |
| 四月 | **し**がつ | April |
| 五月 | ごがつ | May |
| 六月 | ろくがつ | June |
| 七月 | **しち**がつ | July |
| 八月 | はちがつ | August |
| 九月 | **く**がつ | September |
| 十月 | じゅうがつ | October |
| 十一月 | じゅういちがつ | November |
| 十二月 | じゅうにがつ | December |

| 5th | 五日 | **いつか** |
| 6th | 六日 | **むいか** |
| 7th | 七日 | **なのか** |

### 曜日 (ようび) — Days of the Week

| Kanji | Reading | Day |
|---|---|---|
| 日曜日 | にちようび | Sunday |
| 月曜日 | げつようび | Monday |
| 火曜日 | かようび | Tuesday |
| 水曜日 | すいようび | Wednesday |
| 木曜日 | もくようび | Thursday |
| 金曜日 | きんようび | Friday |
| 土曜日 | どようび | Saturday |

Each day uses a natural element from Chinese philosophy (Sun, Moon, Fire, Water, Wood, Gold, Earth).

## Time

### 〜時 (じ) — Hour

Notable readings: 四時 = **よ**じ, 七時 = **しち**じ, 九時 = **く**じ.

Both 12-hour and 24-hour formats are used. 午前 (ごぜん) = AM, 午後 (ごご) = PM.

### 〜分 (ふん/ぷん) — Minute

Rendaku changes ふ → ぷ in several cases:
- 一分 = い**っぷん**, 六分 = ろ**っぷん**, 十分 = じゅ**っぷん**

半 (はん) can replace 三十分 for :30 → 九時半 (9:30).

### 〜秒 (びょう) — Second
Straightforward readings with kango numbers.

## Approximate Time

### 〜くらい / 〜頃 (ころ) — "Around ~"
- 七時**くらい**にそっちに行くね。 — I'll be there **around** seven. (casual)
- 七時**頃**にそちらに伺います。 — I'll be there **around** seven. (formal)

**Key difference:** くらい can mean approximate duration OR point in time. 頃 only means a point in time.
- 四十五分くらい**に**バスに乗る — at around :45 (point in time, に required)
- 四十五分くらいバスに乗る — for about 45 minutes (duration, no に)

### 〜前 (まえ) — "Shortly Before"
- 二時**前**にバス停に来た。 — I came shortly **before** two.

### 〜過ぎ (すぎ) — "Shortly After"
- 二時**過ぎ**にバスが来た。 — The bus came shortly **after** two.
