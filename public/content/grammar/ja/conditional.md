# Conditional Forms (たら、ば、なら、と)

Japanese has four main ways to express "if" or "when." Each has different nuances, and choosing the right one is an important skill. This lesson covers all four patterns with clear examples.

## Overview

| Form | Feeling | Key nuance |
|------|---------|------------|
| たら | If/When (after something happens) | Most versatile, past-based |
| ば | If (hypothetical condition) | Formal, focuses on condition |
| なら | If (it's true that…) | Topic/situation-based |
| と | When/If (automatic result) | Natural consequence, habits |

## たら — "If / When"

たら is the most common and versatile conditional. It is formed from the **past tense (た-form) + ら**.

### Formation

| Type | Dictionary → たら |
|------|-----------------|
| Verb | 行(い)った → 行ったら |
| Verb | 食べた → 食べたら |
| い-adj | 安(やす)かった → 安かったら |
| な-adj | 静(しず)かだった → 静かだったら |
| Noun | 学生だった → 学生だったら |

### Usage & Examples

たら means "if/when X happens, then Y":

- **雨(あめ)が降(ふ)ったら、家(いえ)にいます。** — If it rains, I'll stay home.
- **安かったら、買(か)います。** — If it's cheap, I'll buy it.
- **日本に行ったら、寿司(すし)を食(た)べたいです。** — If I go to Japan, I want to eat sushi.
- **暇(ひま)だったら、映画(えいが)を見(み)ませんか。** — If you're free, shall we watch a movie?
- **宿題(しゅくだい)が終(お)わったら、遊(あそ)びに行きましょう。** — When you finish homework, let's go hang out.

> 💡 たら can also express things that **actually happened** (discovery): 窓(まど)を開(あ)けたら、猫がいた。 — When I opened the window, there was a cat.

## ば — "If (Hypothetical)"

ば focuses on a hypothetical condition and sounds slightly more formal. It emphasizes the **condition itself**.

### Formation

| Type | Rule | Example |
|------|------|---------|
| Group 1 verb | u → e + ば | 書(か)く → 書けば |
| Group 2 verb | drop る + れば | 食べる → 食べれば |
| する | すれば | する → すれば |
| 来(く)る | 来(く)れば | 来る → 来れば |
| い-adj | drop い + ければ | 高(たか)い → 高ければ |
| な-adj | + であれば / なら(ば) | 静か → 静かであれば |
| Noun | + であれば | 学生 → 学生であれば |

### Usage & Examples

- **もっと練習(れんしゅう)すれば、上手(じょうず)になります。** — If you practice more, you'll get better.
- **天気(てんき)がよければ、ピクニックに行きましょう。** — If the weather is good, let's go on a picnic.
- **安ければ、買います。** — If it's cheap, I'll buy it.
- **時間(じかん)があれば、手伝(てつだ)います。** — If I have time, I'll help.
- **早(はや)く起(お)きれば、電車(でんしゃ)に間(ま)に合(あ)います。** — If you wake up early, you'll make the train.

> ⚠️ ば is typically **not** used when the result clause is a request, invitation, or volitional action by the listener. Use たら instead for those.

- ❌ 駅に着(つ)けば、電話してください。
- ✅ 駅に着いたら、電話してください。 — When you arrive at the station, please call me.

## なら — "If (It's True That…)"

なら is used when you take something the other person said (or a known situation) as a starting point for your statement.

### Formation

Simply add **なら** after the plain form of a verb, adjective, or noun:

- 行く**なら** — if you're going to go
- 学生**なら** — if you're a student
- 安い**なら** — if it's cheap
- 静か**なら** — if it's quiet

### Usage & Examples

- **日本に行くなら、京都(きょうと)がおすすめです。** — If you're going to Japan, I recommend Kyoto.
- **魚(さかな)が好(す)きなら、この店(みせ)がいいですよ。** — If you like fish, this restaurant is good.
- **学生なら、割引(わりびき)があります。** — If you're a student, there's a discount.
- **暇なら、手伝ってくれませんか。** — If you're free, could you help me?
- **車(くるま)を買うなら、トヨタがいいと思(おも)います。** — If you're buying a car, I think Toyota is good.

> 💡 なら often implies "I heard that..." or "if what you said is true." It responds to context.

Person A: 明日東京(とうきょう)に行きます。 — I'm going to Tokyo tomorrow.
Person B: 東京に行く**なら**、お土産(みやげ)を買ってきてください。 — If you're going to Tokyo, please buy me a souvenir.

## と — "When / If (Automatic Result)"

と expresses a natural, automatic, or habitual consequence. It implies that Y **always** follows X.

### Formation

Add **と** after the **dictionary form** (plain present):

- 春(はる)になる**と** — when spring comes
- このボタンを押(お)す**と** — if you press this button
- 右(みぎ)に曲(ま)がる**と** — if you turn right

### Usage & Examples

- **春になると、桜(さくら)が咲(さ)きます。** — When spring comes, cherry blossoms bloom.
- **このボタンを押すと、ドアが開きます。** — If you press this button, the door opens.
- **右に曲がると、駅が見えます。** — If you turn right, you'll see the station.
- **お酒を飲(の)むと、顔(かお)が赤(あか)くなります。** — When I drink alcohol, my face turns red.
- **食べすぎると、お腹(なか)が痛(いた)くなります。** — If you eat too much, your stomach will hurt.

> ⚠️ Do **not** use と for requests, invitations, or volitional actions in the result clause.

- ❌ まっすぐ行くと、右に曲がってください。
- ✅ まっすぐ行ったら、右に曲がってください。 — Go straight, then please turn right.

## Comparing the Four Forms

### "If it rains, I'll stay home."

- **雨が降ったら、家にいます。** (たら — most natural for this)
- **雨が降れば、家にいます。** (ば — slightly formal)
- **雨が降るなら、家にいます。** (なら — "if it's going to rain…")
- **雨が降ると、家にいます。** (と — implies you always stay home when it rains)

### Quick Decision Guide

| Situation | Best choice |
|-----------|------------|
| General "if" (safest choice) | たら |
| Hypothetical condition | ば |
| Responding to info / giving advice | なら |
| Automatic/natural result | と |
| Request in result clause | たら |
| Giving directions | と (for description) / たら (for instructions) |

## Negative Conditionals

You can use negative forms with all four:

- **行かなかったら** — if (I) don't go (たら)
- **行かなければ** — if (I) don't go (ば)
- **行かないなら** — if (you're) not going (なら)
- **行かないと** — if (I) don't go (と — implies bad result)

- **薬(くすり)を飲まないと、治(なお)りません。** — If you don't take medicine, you won't get better.
- **勉強(べんきょう)しなければ、テストに受(う)かりません。** — If you don't study, you won't pass the test.
- **急(いそ)がなかったら、遅(おく)れますよ。** — If you don't hurry, you'll be late.
- **行かないなら、連絡(れんらく)してください。** — If you're not going, please let me know.
- **食べないと、元気(げんき)が出(で)ませんよ。** — If you don't eat, you won't have energy.

## Summary

| Form | Formation | Best for |
|------|-----------|----------|
| たら | た-form + ら | General "if/when," requests |
| ば | e-sound + ば | Hypothetical conditions |
| なら | plain form + なら | Responding to information |
| と | dictionary form + と | Automatic/natural results |

Start with たら as your default — it works in the most situations. As you gain experience, you will develop a feel for when the others are more natural.
