# こそあど言葉 (Ko-So-A-Do Words) — Overview

> **Source:** Content from [Tofugu](https://www.tofugu.com/japanese-grammar/kosoado/). For personal study.

こそあど言葉 are sets of Japanese words beginning with こ, そ, あ, or ど. They refer to things, people, places, and manners based on **relative distance** from the speaker and listener.

## The こそあど System

| Prefix | Meaning | Distance |
|---|---|---|
| こ | This (close to speaker) | Near |
| そ | That (near listener / mid-distance) | Away |
| あ | That over there (far from both) | Distant |
| ど | Question word | ? |

## Complete こそあど Chart

| Category | こ (close) | そ (away) | あ (distant) | ど (question) |
|---|---|---|---|---|
| Things | これ/この〜 | それ/その〜 | あれ/あの〜 | どれ/どの〜 |
| People | こいつ | そいつ | あいつ | どいつ |
| Place | ここ | そこ | あそこ | どこ |
| Direction | こちら/こっち | そちら/そっち | あちら/あっち | どちら/どっち |
| Manner | こう/こんな | そう/そんな | ああ/あんな | どう/どんな |

## Three Grammar Types

### Pronouns (stand alone like nouns)
- **これ**はネコです。 — **This** is a cat.
- **ここ**に座ってください。 — Please sit **here**.

### Noun Modifiers (attach to nouns)
- **その**ネコは白い。 — **That** cat is white.
- **どの**山？ — **Which** mountain?

### Adverbs (modify verbs/adjectives)
- **ああ**するのはどう？ — How about doing it **like that**?
- **こう**書いてください。 — Please write it **this way**.

## Key Sets Explained

### これ / それ / あれ / どれ
"This one / That one / That one over there / Which one?" — for identifying things.

### この / その / あの / どの
Same distance system but used **before a noun**: この山 (this mountain), どの山? (which mountain?).

### ここ / そこ / あそこ / どこ
Location words: "Here / There / Over there / Where?"

### こちら / そちら / あちら / どちら
Polite direction/place/person reference. こっち/そっち/あっち/どっち are casual versions.

### こんな / そんな / あんな / どんな
"This kind of / That kind of / That kind of (over there) / What kind of?" — describe manner before nouns.

### こう / そう / ああ / どう
"This way / That way / That way (over there) / How?" — adverbs before verbs/adjectives.

## Distance Rules

**When speaker and listener are side by side:**
- こ = close to both
- そ = medium distance
- あ = far from both

**When speaker and listener face each other:**
- こ = close to speaker
- そ = close to listener (or far from speaker)
- あ = far from both
