# Honorific & Humble Speech (敬語)

Japanese has a formal speech system called **敬語(けいご)** that shows respect for others and humility about yourself. It is essential for business, customer service, and speaking with elders or superiors.

## Three Types of 敬語

| Type | Japanese | Purpose |
|------|---------|---------|
| 尊敬語(そんけいご) | Honorific | Elevates the other person's actions |
| 謙譲語(けんじょうご) | Humble | Lowers your own actions |
| 丁寧語(ていねいご) | Polite | General politeness (です/ます) |

> 💡 You already know 丁寧語 — it's the です/ます form you have been using! This lesson focuses on 尊敬語 and 謙譲語.

## 丁寧語 (Polite Language) — Quick Review

This is the most basic level of politeness:

- **行きます** instead of 行く
- **食べます** instead of 食べる
- **です** instead of だ
- **ございます** (very polite form of あります)

- **お水(みず)はありますか。** — Is there water? (polite)
- **お水はございますか。** — Is there water? (very polite)

## 尊敬語 (Honorific Language)

Used to describe the **actions of someone you respect** (boss, customer, teacher, elder).

### Pattern 1: お + stem + になる

For many verbs, use **お + verb stem + になる**:

| Plain | Honorific |
|-------|-----------|
| 読む | お読みになる |
| 書く | お書きになる |
| 待つ | お待ちになる |
| 帰る | お帰りになる |
| 使(つか)う | お使いになる |

- **先生はもうお帰りになりました。** — The teacher has already gone home.
- **社長(しゃちょう)はこの本をお読みになりましたか。** — Has the president read this book?
- **少々(しょうしょう)お待ちになってください。** — Please wait a moment.
- **お客様(きゃくさま)はもうお食事(しょくじ)になりましたか。** — Has the customer already eaten?
- **こちらのペンをお使いになりますか。** — Would you like to use this pen?

### Pattern 2: Special honorific verbs

Some common verbs have dedicated honorific forms:

| Plain | Honorific | Meaning |
|-------|-----------|---------|
| 行く / 来る / いる | いらっしゃる | go / come / be |
| 食べる / 飲む | 召(め)し上(あ)がる | eat / drink |
| 言(い)う | おっしゃる | say |
| する | なさる | do |
| 見る | ご覧(らん)になる | look / see |
| 知(し)っている | ご存知(ぞんじ) | know |
| くれる | くださる | give (to me) |

- **先生はいらっしゃいますか。** — Is the teacher here?
- **何(なに)を召し上がりますか。** — What would you like to eat?
- **部長(ぶちょう)がおっしゃいました。** — The department head said (it).
- **どうぞご覧になってください。** — Please have a look.
- **先生はご存知ですか。** — Does the teacher know?

### Pattern 3: 〜れる / 〜られる (Passive as honorific)

The passive form can also be used as a lighter honorific:

- **先生は何時(なんじ)に来られますか。** — What time will the teacher come?
- **社長は出張(しゅっちょう)に行かれました。** — The president went on a business trip.
- **部長はもう帰られました。** — The department head has already left.

This is less formal than the お〜になる pattern or special verbs.

## 謙譲語 (Humble Language)

Used to describe **your own actions** when speaking to someone you respect. You lower yourself to show respect.

### Pattern 1: お + stem + する

| Plain | Humble |
|-------|--------|
| 持(も)つ | お持ちする |
| 送(おく)る | お送りする |
| 待つ | お待ちする |
| 伝(つた)える | お伝えする |
| 届(とど)ける | お届けする |

- **お荷物(にもつ)をお持ちします。** — I'll carry your luggage.
- **メールをお送りしました。** — I sent you an email.
- **入り口(いりぐち)でお待ちしています。** — I'll be waiting at the entrance.
- **伝言(でんごん)をお伝えします。** — I'll pass on the message.
- **ご自宅(じたく)にお届けします。** — I'll deliver it to your home.

### Pattern 2: Special humble verbs

| Plain | Humble | Meaning |
|-------|--------|---------|
| 行く / 来る | 参(まい)る | go / come |
| いる | おる | be |
| 食べる / 飲む | いただく | eat / drink |
| 言う | 申(もう)す | say |
| する | いたす | do |
| 見る | 拝見(はいけん)する | look / see |
| 知っている | 存(ぞん)じている | know |
| あげる | さしあげる | give |
| もらう | いただく | receive |
| 聞(き)く | うかがう | ask / hear |
| 会(あ)う | お目(め)にかかる | meet |

- **明日(あした)参ります。** — I will come tomorrow. (humble)
- **田中と申します。** — My name is Tanaka. (humble)
- **お茶(ちゃ)をいただきます。** — I'll have some tea. (humble)
- **資料(しりょう)を拝見しました。** — I looked at the documents. (humble)
- **私が案内(あんない)いたします。** — I will guide you. (humble)

## Common Business Expressions

| Situation | Expression |
|-----------|-----------|
| Answering the phone | お電話(でんわ)ありがとうございます |
| Self-introduction | 〜と申します |
| Before eating | いただきます |
| Leaving work | お先(さき)に失礼(しつれい)します |
| Welcoming a customer | いらっしゃいませ |
| Apologizing | 申し訳(もうしわけ)ございません |
| Understanding | かしこまりました (understood/certainly) |
| Declining | ご遠慮(えんりょ)させていただきます |

- **お忙(いそが)しいところ、申し訳ございません。** — I'm sorry to bother you when you're busy.
- **かしこまりました。すぐにお持ちいたします。** — Understood. I'll bring it right away.
- **ご確認(かくにん)いただけますでしょうか。** — Could you please confirm?
- **お先に失礼します。** — Excuse me for leaving before you.
- **少々お待ちくださいませ。** — Please wait a moment.

## Honorific vs Humble — Side by Side

| Action | Honorific (their action) | Humble (my action) |
|--------|------------------------|-------------------|
| Go/Come | いらっしゃる | 参る |
| Be | いらっしゃる | おる |
| Eat/Drink | 召し上がる | いただく |
| Say | おっしゃる | 申す |
| Do | なさる | いたす |
| See | ご覧になる | 拝見する |
| Give | くださる | さしあげる |
| Receive | — | いただく |
| Know | ご存知 | 存じている |

## Common Mistakes

### 1. Using honorifics for yourself

- ❌ 私がいらっしゃいます。 (using honorific for yourself)
- ✅ 私が参ります。 (humble for yourself)

### 2. Using humble forms for others

- ❌ 先生が申しました。 (using humble for teacher)
- ✅ 先生がおっしゃいました。 (honorific for teacher)

### 3. Double honorifics

- ❌ お召し上がりになる (double honor)
- ✅ 召し上がる (one honorific is enough)

## When to Use 敬語

| Situation | Level |
|-----------|-------|
| Friends, family | Plain / casual |
| Strangers, colleagues | 丁寧語 (です/ます) |
| Boss, teacher, elders | 尊敬語 + 謙譲語 |
| Customers, formal business | Full 敬語 |

## Practice Tips

- Start with the special verb pairs (いらっしゃる/参る, おっしゃる/申す, etc.)
- Listen to customer service interactions for real-world examples
- Don't worry about perfecting 敬語 immediately — even native speakers sometimes struggle with it!
- Focus on the most common patterns: いただく, させていただく, and the お〜になる/お〜する patterns
